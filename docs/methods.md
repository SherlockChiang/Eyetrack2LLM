# Methods

The analysis models destination allocation conditional on a retained transition already being forward, within sentence, and within line; it does not model whether movement is forward. This observational destination endpoint is not the reader's intended saccade target, is not restricted to first pass, and can include rereading or regression-after-forward contexts. Fixations are converted conservatively, candidate destinations are defined before fitting, and nuisance expectations are estimated with text-level cross-fitting. Raw Pearson residuals are then evaluated for fixed-sample half-partition agreement, alignment with the generating feature in simulation, fixed-split auxiliary alignment, and cross-corpus transfer.

The primary agreement analysis uses non-overlapping reader halves and repeated dependent partitions of one fixed reader/text sample; its IQR is not a confidence interval or population reliability estimate. The common core includes target length, frequency, punctuation, token distance, sentence position, and dependency features. It excludes cloze predictability so Provo and ZuCo share a feature universe, and excludes fixation-specific physical launch/landing geometry, width, and visual angle. The four smaller/richer candidate predictive specifications describe bounded controls; a richer specification is not asserted to remove all nuisance or to be more adequate. Auxiliary experiments use architecturally matched conditions with the same initialization and optimization budget. The MLM relation endpoint is an MLM-trained adapter plus an untrained random relation head: it receives no data gradient, although default AdamW decay applies. Transfer uses fixed Provo-trained checkpoints on ZuCo without target-driven checkpoint selection.

Linguistic and oculomotor variables may affect both entry into the retained-transition risk set and destination allocation conditional on entry. Associations within this selected risk set therefore need not equal associations in the full fixation process.

Reusable implementations are in `src/eyetrack2llm/`; exact public command lines are in [reproducibility.md](reproducibility.md).
