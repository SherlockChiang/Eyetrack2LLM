# Limitations

The estimand is specification- and risk-set-dependent, not an unconditional next-fixation model, semantic distance, or causal effect. Reliability is necessary but does not establish latent recovery. Optimization seeds are not independent data replicates, the Provo learnability result uses one fixed ten-text test split, and ZuCo transfer changes corpus, readers, task, texts, and layout simultaneously.

The study does not identify individual traits, parafoveal processing, cognitive-model parameters, universal transport failure, or downstream efficiency gains. Restricted participant-level corpora and checkpoints are not redistributed. See the manuscript's Methods, Results, and Limitations sections for the complete scope and uncertainty statements.
